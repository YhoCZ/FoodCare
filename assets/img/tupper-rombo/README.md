# FoodCare Features Images
Place your transparent PNG images for the features section here.
The files should be named:
- `multicanal.png` (Registro Multicanal)
- `tuppers.png` (Control de Tuppers)
- `ia.png` (IA que Decide)
- `alertas.png` (Alertas Contextuales)

If an image is not present, the system will automatically fall back to showing the corresponding emoji.
